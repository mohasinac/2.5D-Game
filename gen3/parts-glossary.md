## BURST SYSTEM EVOLUTION OVERVIEW

The Burst Generation (gen3) ran 2016–2023 across 7 distinct sub-systems. Each introduced new structural layers and gimmick components on top of the base Energy Layer + Forge Disc + Performance Tip skeleton.

```
ERA TIMELINE:

S1 Standard Burst (2016)
  Energy Layer + Forge Disc + Performance Tip

S2 God Layer (2017–18)
  God/Energy Layer + God Chip + Forge Disc + Performance Tip

S3 Cho-Z (2018–19)
  Cho-Z Layer (metal-embedded) + [Layer Weight] + Forge Disc + Performance Tip

S4 Gatinko / GT (2019–20)
  Layer Base + Gatinko Chip + [Forge Disc + Frame] + Performance Tip

S5 Superking (2020–21)
  Superking Layer (single or double) + Forge Disc + [Frame] + Performance Tip

S6–S7 Dynamite Battle (2021–22)
  DB Core + Armor (Burst) + Blade (Burst) + Chassis (Single/Double) + Performance Tip

S8 BU / Burst Ultimate (2022–23)
  Ring + Layer Base + Gatinko Chip + Forge Disc + Performance Tip
```

DRIVER UPGRADE PATH (Burst generation):

## FRAMES (S4 Gatinko through S5 Superking+)

Frames are additional outer rings that mount over the Forge Disc in GT and Superking era builds. They add mass to the disc stack and can modify the outer contact geometry. Frames were introduced in S4 and became standard in S5.

| Frame | ~Weight | Profile | Effect | Era | Notes |
|-------|---------|---------|--------|-----|-------|
| Outer | ~5g | Standard circular outer | Adds perimeter mass; minimal profile change | S4 | Most common frame; pure weight addition at disc perimeter. |
| High Outer | ~6g | Taller outer profile | Raises contact geometry height; aggressive angle | S4–S5 | Taller variant of Outer; shifts where opponents contact the bey. |
| Under | ~5g | Low-profile under-disc frame | Adds underside mass; low contact geometry | S4 | Low-riding frame; reduces vulnerability to upper attacks. |
| Glaive | ~6g | Blade-edge protrusions | Secondary smash attack contacts on disc | S5 | Glaive-shaped protrusions extend attack reach of the disc stack. |
| Knuckle | ~5g | Knuckle protrusion frame | Three-knuckle attack contact points | S4–S5 | Adds attack protrusions to disc profile; attack builds. |
| Turn | ~5g | Turning geometry frame | Assists circular movement | S5 | Smooth circular profile assists tornado and orbital movement. |
| Wall | ~6g | Wall-contact profile frame | Defense wall contact geometry | S5 | Wall-profiled frame adds defensive contact geometry to disc stack. |
| Xtend+ Frame | ~7g | Integrated frame for Xtend+ driver | Part of Xtend+ driver assembly | S5 | Xtend+ Driver integrates its frame — it is both frame and driver simultaneously. Not a standalone frame. |

## DB SYSTEM COMPONENTS (S6–S7 Dynamite Battle)

The DB System replaced the traditional Energy Layer + Forge Disc stack with a five-piece modular assembly.

```
DB SYSTEM ASSEMBLY (top to bottom):

  ┌────────────────────────────────────┐
  │  BLADE (Burst)                     │
  │  Attack/contact geometry           │
  │  spirit motif absent here          │
  ├────────────────────────────────────┤
  │  ARMOR (Burst)                     │
  │  Spirit motif + outer armor skin   │
  │  Clips over DB Core                │
  ├────────────────────────────────────┤
  │  DB CORE                           │
  │  Structural center shaft           │
  │  Holds Armor + Blade               │
  ├────────────────────────────────────┤
  │  CHASSIS (Single or Double)        │
  │  Replaces Forge Disc               │
  │  Houses the burst pawl mechanism   │
  ├────────────────────────────────────┤
  │  PERFORMANCE TIP                   │
  └────────────────────────────────────┘
```

| Component | ~Weight | Role | Notes |
|-----------|---------|------|-------|
| DB Core | ~8g | Structural center | Universal across all DB beys; holds Armor and Blade assembly |
| Armor (Burst) | ~10–14g | Spirit motif + outer shell | Carries bey's spirit theme; different Armors = different bey identities |
| Blade (Burst) | ~12–18g | Attack/defense contact geometry | Provides actual contact zones for battle; heavier Blades = stronger smash |
| Single Chassis | ~16g | Burst pawl + disc (light) | Attack-optimized Chassis; lighter weight, higher burst rate, more aggressive movement |
| Double Chassis | ~20g | Burst pawl + disc (heavy) | Defense/stamina Chassis; heavier, more stable, lower burst rate |

## GEARS (Gatinko / GT Era — S4)

Gears are mode-switch accessories that attach to certain Forge Discs or Frames in the GT era. They change the mode of specific Layer Bases that support gear engagement.

| Gear | Code | Function | Movement Effect |
|------|------|----------|-----------------|
| D Gear | D | Defense Gear — triggers defense mode on compatible Layer Base | Locks layer in low-resistance position; reduces burst risk; increases stamina |
| F Gear | F | Flat/Friction Gear — activates attack flat mode | Forces layer into aggressive flat contact mode; increases attack aggression |
| V Gear | V | Variable Gear — enables variable mode switching | Allows automatic mode switching between attack and defense based on spin rate |
| VS Gear | VS | Versus Gear — dual attack/defense switch | Engages simultaneous contact in both attack and defense geometries |
| S Gear | S | Stamina Gear — enhances stamina mode | Locks layer in optimal stamina position; reduces friction losses |
| H Gear | H | High Gear — enables high-mode engagement | Raises layer position to high mode for aggressive upper-attack contact angle |
| L Gear | L | Low Gear — enables low-mode engagement | Lowers layer to low defensive position; reduces burst risk and KO vulnerability |

## SPECIAL ACCESSORIES

### Layer Weight (Cho-Z Era — S3)

A metal clip-on accessory that attaches to the underside of compatible Cho-Z Layers. Adds ~5–7g of zinc alloy weight directly to the layer, dramatically increasing smash attack power and rotational inertia without increasing disc weight.

| Property | Value |
|----------|-------|
| Material | Zinc alloy (metal) |
| ~Weight added | 5–7g |
| Attachment | Clips to underside of Cho-Z Layer |
| Compatibility | Cho-Z era layers only |
| Effect | Increased smash attack, lower burst risk, heavier assembly |

### Gatinko Chip (S4 GT Era)

The replaceable spirit chip that slots into the center of a GT Layer Base. Carries the bey's spirit motif and the character/blader identity. Can be swapped between compatible Layer Bases for custom combinations.

| Property | Value |
|----------|-------|
| Material | ABS plastic (some premium metal-foil versions) |
| ~Weight | ~3–5g |
| Attachment | Snap-fit into Layer Base center socket |
| Compatibility | GT-era Layer Bases (and BU-era Layer Bases) |
| Chip types | Standard Chip, Premium Chip (metal-foil face) |
| Game engine | `ContactZoneSpec.zoneType: gatinko_armor` |

### Burst Blade (DB Era — S6)

The contact-geometry component of the DB System. Not to be confused with gen4's Blade (BX). The DB Burst Blade snaps over the DB Armor and provides the actual attack/defense profile for battle.

### Layer Base (GT Era — S4 and BU Era — S8)

The main structural body of a GT-era Beyblade's layer section. Pairs with a Gatinko Chip. Different Layer Bases provide different attack, defense, or stamina geometries and support different gear gimmicks.
