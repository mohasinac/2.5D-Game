# NAME_CONFLICTS.md — Regional Naming Notes

This file records regional naming differences found across case study files CS1–CS9.
These are NOT true conflicts (the parts are physically identical); they are cross-regional name equivalencies.

---

## TT (Takara Tomy) vs Hasbro Regional Name Pairs

| TT Name | Hasbro Name | System | Notes |
|---------|-------------|--------|-------|
| Shinobi Saramanda | Ninja Salamander | Zero-G (Chrome Wheel / Crystal/Element Wheel) | Same Chrome Wheel; Hasbro renamed both blader title and element wheel |
| Eonis | Ionis | MFB (Metal Fight Beyblade) — Energy Wheel/Ring | Same Energy Wheel; Hasbro spelling variant of TT name |

---

## Notes

- All case study entries use the **Takara Tomy** name as the canonical reference, consistent with the Burst Series Scope (Hasbro excluded from Gen 3 / 4D onward per project scope).
- For Plastics / MFB generations where both TT and Hasbro products co-exist, TT names are used throughout; Hasbro names noted above for lookup purposes only.
- No physical part conflicts were identified — all indexed components are uniquely identified by case number and TT product name.
