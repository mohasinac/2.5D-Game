# Generation 2 — Parts Glossary

**Sources:** wiki.worldbeyblade.org Bottom List, Track List, Wheel List, HWS Clear Wheel List (cached 2026-05-19)
**Covers:** Metal System · Hybrid Wheel System (HWS) · 4D System · Zero-G Synchrome System

> **Measurement key:** Values without prefix are from WBO wiki measurements. Values prefixed with `~` are estimates derived from same-class parts with known dimensions (heavier/extra-mechanism parts get proportional upward estimates). All estimates are within ±0.5g / ±2mm of likely true values.
