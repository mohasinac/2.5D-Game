## SECTION A — BURST SYSTEM ARCHITECTURE [FACT — Beyblade Fandom Wiki, Burst System article]

### A1. How Burst Parts Are Named

| Sub-System | Series Name | Top Part | Middle Part | Bottom Part |
|-----------|------------|----------|-------------|-------------|
| Burst System (original) | Beyblade Burst (2015) | Energy Layer | Forge Disc | Performance Tip / Driver |
| God Layer System | Beyblade Burst Evolution (2017) | God Chip + God Layer | Core Disc + Frame | Driver |
| Cho-Z Layer System | Beyblade Burst Turbo (2018) | Cho-Z Chip + Cho-Z Layer + Cho-Z Ring | Core Disc + Frame | Driver |
| Gatinko Layer System | Beyblade Burst Rise (2019) | Gatinko Chip + Gatinko Layer | Disc + Frame + Weight | Driver |
| Sparking System | Beyblade Burst Surge (2020) | Chip + Ring | Double / Single Chassis | Driver |
| Dynamite Battle System | Beyblade Burst QuadDrive / DB (2021) | Blade + Core | Disc | Driver |
| Ultimate Layer System | Beyblade Burst Ultimate (2022–2023) | Blade + Core | Disc | Driver |

**The Driver / Performance Tip is always the bottom spinning contact part.** It is the Burst equivalent of MFB's Performance Tip. "Driver" is the standard modern term; earlier Burst products said "Performance Tip."

## SECTION C — DRIFT vs BEARING DRIFT — FULL DISAMBIGUATION

### C1. Three Separate Concepts

| Term | Category | Definition |
|------|----------|-----------|
| **Drift** (physics) | General physics concept | A motion state where an object slides across a surface **maintaining its current speed and velocity vector** despite lateral forces. Momentum is conserved while sliding — the object does not decelerate to a stop. Like a car drifting. In Beyblade tip physics: a drift tip slides when pressed laterally rather than gripping and redirecting. |
| **Drift** (Burst driver) | Official Burst part | A chunky, wide apple/pear-shaped body (the pear rotates independently) sitting on a small free-spinning plate at the bottom. NO ball bearings — free rotation via plastic pivot only. The plate slides under lateral pressure. Stamina-oriented. Shape CORRECTED from prior "conical sharp" description — confirmed by user. |
| **Bearing Drift** (Burst driver) | Official Burst part | Debuted B-203 (October 2022, Ultimate Fusion DX Set). Upgraded Drift with ball bearings in the free-spinning shaft. Wide octagonal guard body. Conical sharp tip. Abbreviation: **BD** (no colon — unrelated to MFB B:D). |

### C2. Bearing Drift Physical Description [FACT — from Beyblade Fandom Wiki, official product description]

> *Official description*: "A Defense Type Driver with large guard parts that can generate centrifugal force with a free-spinning tip and has a low overall height for high stability and contains a built-in bearing for high Stamina."

| Attribute | Detail |
|-----------|--------|
| Body shape | Wide, round, octagonal silhouette |
| Tip shape | Low-angled free-spinning conical sharp |
| Bearing type | Internal ball bearings in tip shaft |
| Height | Low overall height for stability |
| Movement | Near-stationary; tip slides under pressure |
| Predecessor | Drift driver |
| Variants | Metal Bearing Drift (MBD) — metal tip; same mechanism |

### C3. Confusion Table

| Pair | Differentiation |
|------|----------------|
| Drift (physics) vs Drift (driver) | Physics = general sliding behavior. Driver = a specific Burst part exhibiting that behavior. |
| Drift (driver) vs Bearing Drift | Drift = no ball bearings. Bearing Drift = ball bearings added (2022). |
| Bearing Drift (BD) vs B:D (MFB) | BD no colon = Burst driver. B:D with colon = MFB 4D Bottom "Bearing Drive." Different generations. |
| Bearing (driver) vs Bearing Drift | Bearing = simpler, bearing-only. Bearing Drift = bearing + drift + wide octagonal guard body. |

## SECTION E — BURST LAYER CHIP SHAPES [FACT — user-specified canonical shapes]

The identity chip at the center of a Burst beyblade (God Chip, Cho-Z Chip, Gatinko Chip, etc.) has these canonical shapes:

| Shape | Description | Notes |
|-------|-------------|-------|
| **Triangular nut (3-sided)** | Equilateral triangle; 3 locking points | Fundamental to Burst mechanic: 3 lock tabs release sequentially under impact [FACT] |
| **Hexagonal nut** | Regular hexagon; 6 sides | More lock points = generally higher burst resistance [INFERENCE] |
| **Circle disc** | Round disc | Found in some simplified designs [INFERENCE] |
| **Rectangle with inscribed circle** | Rectangle where circle radius > rectangle half-width; lens/stadium outline | Found in some chip faces [INFERENCE] |

**3-point burst lock** [FACT]: The 3 lock tabs on the Layer Base correspond to the triangle's 3 corners. This is how the Burst elimination mechanic works. Chip shape and burst resistance correlate [INFERENCE] but spring strength, material, and layer profile also factor in.
