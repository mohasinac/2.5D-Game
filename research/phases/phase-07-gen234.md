### Shared Patterns Discovered (Batch 7F)

- **4D System gimmick:** Multi-piece metal frame + core wheels (Big Bang, Wing/Cosmic, Flash, Blitz, Kreis, Scythe, Death, Jade, Duo, Phantom, Fang) → `mode_switch` gimmick is universal for this system.
- **F:D tip** is the canonical `final_drive` gimmick — RPM-triggered auto mode switch.
- **Basalt/Twisted FW** at ~50g is the engine's heaviest inertial defense example; maps to `heavy_wheel` + very high staminaPts.
- **BD145** free-spin disc maps to `free_spin` gimmick on track.
- **Dual-spin beys** (Gravity, Variares, Flame Pegasus) get `rotation_reverse` gimmick.
- Mass range for competitive MFB: ~32–54g assembled.

- **Burst system mass range:** ~32–42g assembled (lighter than gen2 due to plastic emphasis).

- **Burst-era burst resistance** is a Colyseus state property (burst threshold), not a separate gimmick.

### Shared Patterns Discovered (Batch 7M)

- **BX Xtreme Dash** is the defining gen4 gimmick — maps to `xtreme_dash`. All rubber-flat Bits (F, LF, R, FB, V) and gear-assisted Bits (GF, GP, A, H) qualify.

- **DB Bit** (WizardRod) = `free_spin` + `bearing_zombie` — the zombie combination; banned April 2025.
- **0-tooth Ratchet** (GhostCircle) = immunity to Burst → `free_spin` maps this conceptually.

- **P Bit dual-mode** = `mode_switch`; **TP Bit auto-cycle** = `mode_switch` + `xtreme_dash`.
- **Q Bit asymmetric** = `weight_shift` (wobble pattern from intentional asymmetry).
- Mass range: 37–50g for standard BX; CX beys at ~50g due to Lock Chip + Assist Blade.
